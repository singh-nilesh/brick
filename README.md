> Brick is a React Native app that helps users build structured, long-term career roadmaps.
It combines AI-generated learning plans, habit tracking, task management, and curated references to streamline skill-building and career growth.

## What Problem Does It Solve?
- Students often struggle with:
- No clear path to reach career goals
- Unstructured skill-building and lack of consistent habits
- Difficulty tracking progress over months/years
- Wasting time gathering resources
- Manually scheduling tasks across apps

### Brick addresses these challenges with an all-in-one system.
