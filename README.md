# Brick

Conquer Your Ambitions Brick by Brick

## Overview

Brick is a modern productivity app designed to help individuals stay on track with their daily, monthly, and yearly goals. Built with AI-driven insights, the app empowers users to achieve their career ambitions step by step while maintaining daily consistency and self-awareness. Whether you're building skills, tracking tasks, or journaling for self-reflection, Brick offers the tools to grow your potential.


## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```
